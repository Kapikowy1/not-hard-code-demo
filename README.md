not-hard-code

not-hard-code is a demo web application that presents a platform for online programming courses.

The application was built using React and TypeScript, with Vite used as the build tool to provide a fast development environment and quick project setup.

The interface is responsive, so it works properly on both desktop and mobile devices.

This project is a demo version, which means it only includes the frontend part. There is no backend or database.

The main goal of the project is to demonstrate:

building applications with React + TypeScript

using Vite as a modern build tool

creating responsive user interfaces

organizing components in a Single Page Application (SPA)
